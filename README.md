
# TellMe - A platform for sharing stories

TellMe is a modern and lightweight platform for sharing stories about your travels, work, and ideas. It's a place where you can connect with other people, share your experiences, and get inspired.

The backend is a [Strapi](https://strapi.io/) instance, a leading open-source headless CMS, and the frontend is built with [Astro](https://astro.build/), a modern and performant static site generator. The combination of these two technologies allows for a flexible, powerful, and easy-to-use platform.

## ‚ú® Features

- **Share your stories:** Write and publish your stories with a simple and intuitive editor.
- **Connect with others:** Read and comment on stories from other users.
- **Discover new ideas:** Get inspired by stories about travel, work, and creativity.
- **Lightweight and performant:** The frontend is a static site, which means it's fast, secure, and easy to deploy.
- **Easy to customize:** The backend is a Strapi instance, which means you can easily customize the data structure and add new features.

## üöÄ Getting started

### Prerequisites

- [Node.js](https://nodejs.org/en/) (v14 or higher)
- [npm](https://www.npmjs.com/) (v6 or higher)
- [git](https://git-scm.com/)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/tellme.git
```

2. Install the dependencies for the backend:

```bash
cd backend
npm install
```

3. Install the dependencies for the frontend:

```bash
cd ../frontend
npm install
```

4. Install the dependencies for the second frontend:

```bash
cd ../frontend-v2
npm install
```

### Usage

1. Start the backend server:

```bash
cd ../backend
npm run develop
```

2. In a new terminal, start the frontend development server:

```bash
cd ../frontend
npm run dev
```

3. In a new terminal, start the second frontend development server:

```bash
cd ../frontend-v2
npm run dev
```

The backend will be running at `http://localhost:1337` and the frontends at `http://localhost:4321` and `http://localhost:4322`.
